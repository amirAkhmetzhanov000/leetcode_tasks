# [21. Merge Two Sorted Lists](https://leetcode.com/problems/merge-two-sorted-lists/)

---

## Описание

Вам даны головы двух отсортированных связанных списков `list1` и `list2`.

Объедините два списка в один **отсортированный** список. Список должен быть создан путем связывания узлов исходных списков.

Верните *голову объединенного связанного списка*.

### Ограничения

* Количество узлов в обоих списках: $[0, 50]$.
* Значение узла: $-100 \le \text{Node.val} \le 100$.
* Оба списка отсортированы по **неубыванию**.

---

## Примеры

**Example 1:**

> **Input:** `list1 = [1,2,4], list2 = [1,3,4]`
> **Output:** `[1,1,2,3,4,4]`

**Example 2:**

> **Input:** `list1 = [], list2 = []`
> **Output:** `[]`

**Example 3:**

> **Input:** `list1 = [], list2 = [0]`
> **Output:** `[0]`
